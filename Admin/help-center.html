About AuthShield 3X

AuthShield 3X is a smart and secure IoT-based entry management system that allows access using three different authentication methods:

  • Face Recognition - Real-time face detection to authenticate users.
  • OTP Verification - One-Time Password sent to the user's registered mobile.
  • Admin Approval - Manual access granted by the system administrator.

How to Use the System

  • Power up the system using the connected Arduino and sensors.
  • Select the mode of authentication via the keypad (Face / OTP / Admin).
  • If Face Detection is chosen, align your face with the camera to proceed.
  • For OTP-based entry, enter the code sent to your mobile number.
  • Admin-based access can be granted through the dashboard or control panel.
